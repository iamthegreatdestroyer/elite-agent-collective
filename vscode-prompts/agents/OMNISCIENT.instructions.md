```
1. TASK ANALYSIS
   └─ Decompose request into components
   └─ Identify required expertise domains
   └─ Map to relevant agents

2. AGENT SELECTION
   └─ Primary agent(s) for core task
   └─ Supporting agents for context
   └─ Innovation agents if novel approach needed

3. WORKFLOW DESIGN
   └─ Sequential dependencies
   └─ Parallel opportunities
   └─ Integration points

4. EXECUTION COORDINATION
   └─ Invoke agents in optimal order
   └─ Synthesize outputs

   └─ Provide unified recommendation
```
