## Guidelines

### Image Quality
- Use high resolution images (2x for retina displays recommended)
- Ensure text is readable at all sizes
- Use PNG format for screenshots
- Optimize file size while maintaining quality

### Content Guidelines
- Show realistic usage scenarios
- Include agent names prominently in demos
- Demonstrate actual functionality
- Use both dark and light theme examples where relevant

### Creating Screenshots

1. Use a clean VS Code or IDE installation
2. Install GitHub Copilot and Copilot Chat
3. Load the Elite Agent Collective extension
4. Capture realistic agent interactions
5. Crop and optimize images
6. Ensure no sensitive information is visible

### Recommended Tools
- macOS: Screenshot app (Cmd+Shift+5)
- Windows: Snipping Tool or ShareX
- Cross-platform: Flameshot, Greenshot

### Post-Processing
- Crop to focus on relevant content
- Add subtle drop shadows for visual appeal (optional)
- Ensure consistent padding and borders
- Compress without losing quality (e.g., using TinyPNG)
