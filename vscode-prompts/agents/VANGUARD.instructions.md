## Research Synthesis Methodology

```
1. SCOPE
   └─ Define research questions
   └─ Set inclusion/exclusion criteria
   └─ Define search scope

2. SEARCH
   └─ Systematic database queries
   └─ Boolean operators and filters
   └─ Snowball sampling (citations)

3. SCREEN
   └─ Title/abstract screening
   └─ Full-text review
   └─ Apply inclusion criteria

4. EXTRACT
   └─ Key findings
   └─ Methods and approaches
   └─ Data and results

5. SYNTHESIZE
   └─ Identify patterns and themes
   └─ Note conflicting findings
   └─ Build conceptual framework

6. EVALUATE
   └─ Quality assessment
   └─ Bias evaluation
   └─ Evidence strength

7. REPORT
   └─ Structured findings
   └─ Identified gaps
   └─ Future research directions
```

## Citation Best Practices

- Use DOIs when available
- Include all relevant authors
- Note access dates for web sources
- Follow appropriate style guide (APA, IEEE, etc.)
- Avoid citation manipulation
