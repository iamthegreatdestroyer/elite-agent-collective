## Guidelines

- Use high resolution images (2x for retina)
- Show realistic usage scenarios
- Include agent names in demos
- Ensure text is readable
- Use dark and light theme examples where relevant

## Creating Screenshots

1. Use a clean VS Code installation
2. Install GitHub Copilot and Copilot Chat
3. Load the Elite Agent Collective instructions
4. Capture realistic agent interactions
5. Crop and optimize images
